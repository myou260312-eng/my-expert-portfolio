Security Policy

Last updated: July 2026

1. Security Architecture

My-Expert-Portfolio runs strictly under isolated browser security permissions. It uses secure sandbox standards to isolate local computations and prevent arbitrary code injection attacks.

2. Permissions Disclosures

My-Expert-Portfolio requests only the minimum required access permissions necessary to perform its core functionalities. It does not monitor external networks or bypass secure connections (HTTPS).

3. Reporting a Vulnerability

If you identify a security vulnerability within the My-Expert-Portfolio project, please do not open a public issue. Instead, report it privately by opening a secure security advisory on our primary GitHub repository or contacting the developer directly.
